# QualiGather Web システム ひな形

## ディレクトリ構成

- `public/` : 公開ディレクトリ（index.php, index.html など）
- `app/` : PHP アプリケーション本体
- `css/` : CSS ファイル
- `js/` : JavaScript ファイル
- `lib/` : 外部ライブラリ
- `assets/` : 画像やフォント等のアセット

## 使い方

- `public/index.html` をブラウザで開くと、ひな形画面が表示されます。
- PHP の処理は `public/index.php` から開始されます。
